# The MESSY Playbook™

**Self-Honesty for People Tired of Being Optimized**

An anti-optimization personal development framework derived from Manufacturing Execution Systems (MES) theory and complexity science. Built and maintained by Robert Merlo / [Live Messy Project](https://livemessy.gumroad.com).

## Deployment

Updates deploy automatically. No Vercel login required.

1. Download updated `.html` file(s)
2. Go to [github.com/Habari2026/messy-playbook](https://github.com/Habari2026/messy-playbook)
3. Click **Add file** → **Upload files**
4. Drag file(s) into the upload box
5. Add a commit message → click **Commit changes**

Vercel detects the commit and deploys within ~60 seconds.
